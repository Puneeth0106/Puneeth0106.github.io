# Puneeth Kumar Amudala — Personal Portfolio

Personal portfolio website built with Angular, deployed on GitHub Pages.

🔗 **Live site:** [puneeth0106.github.io](https://puneeth0106.github.io)

## Built With

- Angular 13
- TypeScript
- Bootstrap
- Font Awesome
- Google Fonts (Syne Mono + Inter)
- GitHub Pages

## Run Locally

```bash
# Install dependencies
npm install -f

# Start dev server
ng serve
# Open http://localhost:4200
```

## Deploy

```bash
ng deploy --base-href="/"
```
